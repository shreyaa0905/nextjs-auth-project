# Next.js Authentication System

## Features
- Signup & Login
- Email Verification
- Forgot Password
- Reset Password
- JWT Authentication
- Protected Routes (Middleware)

## Tech Stack
- Next.js
- MongoDB
- Tailwind CSS
- Nodemailer

## Setup
1. Clone repo
2. Run `npm install`
3. Add `.env` file
4. Run `npm run dev`
